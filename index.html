Final Term Project Proposal:
Python Programmer



For this term project, the Python Programmer path will be chosen. The 10 functions that will be created will take census data of any kind, it will be able to analyze, aggregate, graph or plot the data given to it.

The 10 Functions:

  • Import data and make it read as a geodataframe
  • Aggregate different data together and choose how the data should be aggregated
  • Analyze the data, using calculations to create new data, add this data to the dataframe
  • Create a graph with the data, arguments would be the data, title, axis limits and titles, colours for the data and type of representation (line, dots, etc.), choice of which type of graph (conditional statement)
  • Create a map with the data, arguments would be the data, title, legend limits (if scale)
  • Rename columns
  • Export data, as new csv or as spatial data (geojson, etc.)
  • Check function to make sure that the data you are manipulating is properly formatted and contains the proper data, arguments data and how to check the data (have an if statement to choose different check functions within the main function)
  • Check the projection of the geodata and be able to change it if needed and choose which projection to change it to
  • Fix data so that there are no values that would mess up something (fillna, etc.)
